Cheat Sheet Shortcuts

The most useful Keyboard Shortcuts

for Edge Animate

This cheat sheet will provide you with the most useful keyboard shortcuts for a faster workflow in Edge Animate. Enjoy!

General

  • new Project: CMD/Ctrl + N
  • save Project: CMD/Ctrl + S
  • preview in browser: CMD/Ctrl + Enter
  • code view: CMD/Ctrl + E
  • publish: alt + CMD/Ctrl + S

Tools

  • selection tool: V
  • transform tool: Q
  • clipping tool: C
  • rectangle tool: M
  • rounded rectangle tool: R
  • ellipse tool: O
  • text tool: T
  • hand tool: H
  • zoom tool: Z

Edit

  • undo: CMD/Ctrl + Z
  • redo: CMD/Ctrl + Shift + Z
  • cut: CMD/Ctrl + X
  • copy: CMD/Ctrl + C
  • paste: CMD/Ctrl + V
  • paste transitions from location: Shift + CMD/Ctrl + V
  • paste inverted: Shift + alt + V
  • paste all: Shift + CMD/Ctrl + V
  • duplicate: CMD/Ctrl + D
  • select all: CMD/Ctrl + A

Timeline

  • play/pause: Space
  • return to previous position: Enter
  • next keyframe: CMD/Ctrl + right
  • next keyframe: CMD/Ctrl + left
  • toggle pin: P
  • auto-keyframe mode: K
  • auto-transition mode: X
  • flip playhead and pin: Shift + P
  • remove transition: Shift + Backspace
  • transport playhead:
    • move to end: End
    • move to beginning: Home
  • set trigger: CMD/Ctrl + T
  • set label: CMD/Ctrl + L

Modify

  • Group elements: CMD/Ctrl + G
  • Ungroup elements: Shift + CMD/Ctrl + G
  • Convert to symbol: CMD/Ctrl + Y
  • Move along only one axis: Shift (pressed)
