Getting Creative

Creating with shapes

Options and features

In this section you will learn how to use Edge Animate as a creation tool. You will learn the basics of how to create and modify elements in Edge Animate. Although you will most likely use a bunch of assets like images to create your compositions, it is good to know what Edge Animate itself has to offer. So let's see what we can do here...

Creating elements

The most basic way to create something in Animate is by using elements. You can create new elements by using the according tool bar right above the property inspector. You can use them for example to create basic shapes and text elements.

Let's first have a look at these shapes. There are three options here you can choose from: rectangles, rectangles with rounded corners and ellipses – represented by the according icons in the tool bar. If you click on one of these icons, your cursor turns into a crosshair when entering the stage, allowing you to create a shape. To actually draw the shape onto the canvas, you have to click and drag. The shape is drawn between the point where you start the click and the point where you release the mouse button...as you have most likely already figured out yourself.

Tip: You can create symmetrical shapes like squares and circles when holding the shift key while drawing a shape.

Notice that you can also draw shapes into the empty space around the stage element as well. This way they are positioned outside of the stage. Keep in mind, that only those elements are visible in the browser later, which are placed within the constraints of the stage element (unless the overflow setting is changed to 'visible').

Every time you draw a shape, the new element appears in the elements panel and is selected automatically. Selected elements are highlighted on the stage by being surrounded with blue lines including handles in each corner and along each side. These handles can be used to visually change the dimensions of the element. Your mouse cursor indicates these options when you hover over the lines or handles. The filled blue square in the middle indicates the origin of the element (i.e. its logical "point of reference"), which is important for transformations, for example. If you click anywhere within the blue lines, you can reposition the element while holding the mouse button pressed.

If you take a look at the property inspector, you will notice that changes when an element is selected: It is now displaying the options for the element and no longer those for the stage. All properties shown here apply to the element that is selected. In order to deselect an element, click anywhere outside the blue lines. This will automatically change the scope back to the stage (unless you click on a different element, which is then selected instead).

The property inspector not only provides you with information about the currently selected element, it also allows you to change it. We have already got to know the property inspector in the "Understanding the User Interface" chapter. Now it's time to understand how it works in the creation process.

We already got to know how to position and resize an element visually on the stage. With the property inspector, you can make changes to the values immediately and even type in precise measures. Notice that the element on the stage changes accordingly.

Adding Background color and Gradients

When you draw shapes on the stage, the default color is a light grey. You can change the color by clicking on the left area in the color section of the property inspector. This opens up the color selection panel. Here you can adjust the color with the selector on the left and the bars for hue, lightness and alpha channel on the right or use the input field to set the color in RGBa, Hex or HSLa.

If you want to use a color that already exists somewhere on the stage, you can select a sample with the pipette tool. The corresponding color will then be calculated automatically. Note that when you start making changes, the preview in the top left corner shows both the new and the previous color. This way you can switch back to the original real quick by simply clicking on the old color.

Also quite helpful is the option to save the current color selection by clicking on the "+" button on the right hand side. This allows you to create color swatches and reuse the same colors throughout your composition. Note that the selected color becomes the new default for the next shape you draw.

Similar to the background color is the option to add a gradient to your element. By default, this option is set to "none" (the preview area being crossed with a red line). When you click on this field, the corresponding menu opens up, which is pretty similar to the color selection panel:

To activate a gradient, select linear or radial gradient in the top left bar. With linear gradients you have the option to set the radius, i.e. the direction in which the gradient shall occur. With radial gradients, you can define the point of origin from which the gradient spreads.

For both options, you have the preview bar with the color stops. These color stops indicate the colors used for the gradient, their position determines the percentage (start and end point). To change their color, select one of the color stops and use the color selection panel on the right hand side. You can add furhter color stops by clicking on the preview bar itself. To remove color stops again, drag them out of the preview bar.

Just like with the color selection, you can also save your gradient settings to use them for other elements as well. Simply click on the "+" button and the gradient settings will be stored as a swatch.

The gradient lies ontop of the background image, therefore the background color is only visible if you use an alpha channel for the gradient.

Hint: Gradients are not fully supported by all browsers. A safe bet is to apply a plain background color as well, so that the element will at least be displayed in that color in the corresponding browsers.

Borders

The third option in the color section of the property inspector deals with borders. These borders can have a color, size and style. The default setting for style is none, so that the new element doesn't show a border yet, although it already has a border color:

In order to make the border visible, you have to change the border style to either solid, dotted or dashed. Lastly you can define the thickness of the border with the value on the right. You should keep in mind though that the size of the border will increase the size of the element as a whole.

Transforming elements

Your shapes and elements can further be tweaked and edited with the options in the "Transform" section of the property inspector. You can use scaling, skew elements and rotate them. Lastly you can determine the origin where the transformation relates to.

The first option here is scaling the selected element. This can have effects on the properties for positioning and width or height respectively as well. You can use negative values here as well. So if you set the scaling value to -100% the element is flipped around. Note that by default, both axis are linked. You can unlink the properties by clicking on the link symbol left to the values.

Scaling might not seem to be so powerful on the first look, but it can create better results when applied to images or texts instead of plain shapes. Also keep in mind that you can animate the scaling property as well to create interesting effects.

Skewing shifts two sides of the element according to the value in degrees. Applied to a rectangle, this creates a parallelogram.

As a third option, rotation enables you to rotate the element around the point of origin. The value is presented in degree and defines, how much the element is being rotated. A positive value rotates clockwise, a negative value counter-clockwise. Here is an example of applied transformations:

With the origin property, you can define the point around which the transformation should take place. It is represented by the filled blue square within the selected element. By default it is located in the center of the element. Since the value is relative to the top left corner, this default is 50% on both axis. Note that you can have negative and values greater than 100% as well. This lets you for instance rotate the element around an imaginary fix point outside the element itself:

The best way here is to use basic shapes and play a bit around with the different options to get a feeling for how transformations work.

Clipping

Another way to modify elements in Edge Animate is clipping. As the name already indicates, it is used to hide parts of the element, just like applying a mask. In order to use this feature, you first need to activate it with the switch on the right hand side of the Clip section in the property inspector

Once clipping is enabled, you can use the four values to set the borders for the clipping mask. As usual, the values are related to the top left corner of the element and the dimensions of the mask fit the size of the element. Changing the values changes the size of the blue container although the size of the actual element remains the same. Let's look at an example:

Here we have to circles of the same size. On the right circle, the clipping is enabled and the bottom value reduced to its half (in this case 100px instead of 200px). As you can see, only half of the element is visible, and although the element itself is 200px high, the actual representation on the stage has only half the height. This allows you to create shapes like this semi circle, and it can be quite useful when working with images and for creating animation effects.

Shadows

The shadow property allows you to add a box shadow to your element. This can create spatial effects or even some primitive 3D looks. As with clipping, the shadow options needs to be activated to work. When activated, a shadow is added to the selected element and the correspondant section in the property inspector becomes available:

The default setting here is a drop shadow, that is cast downward and to the right, in a slightly transperant dark grey and a light blur. You can use the icons at the top of this section to switch between drop and inset shadow and then tweak the attributes positioning (X and Y), color, blur and spread. If you click on the color preview for the shadow, the already familiar color selection panel appears, allowing you to choose any kind of color. Especially with shadows, the alpha channel option is interesting, since it allows you to determine the opacity of the shadow.

The blur attribute can only be set with positive values, whereas the positioning and spread can be set to negative values as well, for instance to create a shadow that is cast to the top left side of the corresponding element. Here is an overview of different shadow settings together with the result on the stage:

Tip: Apparently, you can only apply one shadow to each element at the same time. But there is a workaround for that. Since you can use the alpha channel to make elements transperent, it is possible to stack transparent copies of the same element over another. Each of these copies can then have one shadow, so that you basically just add yet another shadow layer with each copy. The example shown below should help to illustrate this effect:

In this case the base element has a simple drop shadow with a blur of 7px. Above that lies a transperent copy with an inset shadow in white (71%), slightly set off down and right (1px in each direction) and with a blur of 21px. Also the spread is increased to 3px. On top of that lies another transparent copy with a grey inset shadow, this time cast in the opposite direction (-3px) and a blur of 8px.

Note that you have to use the alpha channel for the background color, to get a transparent element. When you set the oppacity for the element to 0, the shadow will be invisible as well and therefore the effect is lost.

Another thing that needs to be mentioned here is that we are dealing with box-shadows. As the name already suggests, this means that it is created by the DIV box. Although it may contain rounded corners (due to border radius settings), you will not be able to create a shadow that corresponds to different shapes or character images.

Filters - and why you should be cautious!

Edge Animate allows you to apply CSS filter to element. These filters influence the visual appearance of that element, for instance by inverting the color. Tempting as these options might be, you should be aware of the fact that CSS filters are not yet supported by all modern browsers. Firefox for example will simply ignore these settings. It is therefore highly questionable whether using CSS filters at all is a good idea at the moment. As long as they are not absolutely essential for your composition, you should rather avoid them.

Corners - or: Why the cirlce is a square

Although you have the tools to create three different shapes – rectangles, rectangles with rounded corners and ellipses - all these elements are in fact basically the same: They all are plain DIV elements with a different border radius. The corresponding property is called "Corners" and can be modified as well. Give it a try: Select the rectangle tool and draw a square onto the stage. Now go to the corners section in the property inspector and increase the value. You will notice that it will turn into a rectangle with rounded corners and finally into a circle:

It still makes sense to have the different creation tool options, though, since it is way faster for creating different shapes than having to set the border radius every time you want to create a circular shape. There are also some more options concerning the border radius: Not only can you switch between pixel and percentage based units, you can also define, which part of the element the border radius should be applied to. By default, all four corners are selected. You can use the preview to change the selection. click on the corresponding corner to (de-)select it:

With the buttons above the preview, you can choose how many values you want to set for the border radius: One value for all (1), one for each corner (4), or two values for each corner (8). This last option allows you to create asymmetric curved corners like this:
