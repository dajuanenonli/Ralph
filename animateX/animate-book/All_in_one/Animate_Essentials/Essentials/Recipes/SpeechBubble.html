Recipe: Creation

Creating a speech bubble

with plain elements

This very primitive recipe will show you how you can create a speech bubble with plain elements in Edge Animate. It's nothing fancy, but usable in many occasions, and a good way to start practicing our creation skills in Animate. So let's get to work!

Result

Result

If you combine several elements of the same color and apply transformations, you can create all sorts of shapes out of plain elements. The advantage is that you don't need to import external assets and can work with plain HTML elements.

Challenge: Try to create a simple speech bubble by combining two rectangles. This is what the speech bubble is supposed to look like:

Have you already figured it out? Well, let's take it one step at a time:

First you create the larger rectangle as base. In this example it is one with rounded corners. Then you draw a second rectangle in the shape of a square underneath that first element. Make sure to use the same color for both elements so that they fit into each other.

Now select the smaller rectangle and go over to the Transform section in the elements panel. Use the rotate property to turn the elements by 45 degrees. Then move the element above the larger rectangle, so that it overlaps for the most part. This will add a triangular shape to the large rectangle. Voil√†...

As soon as you de-select the elements it seems as if it is only one shape, that looks like a speech bubble.

Tipp: If you want to make sure that both elements behave like one element together, move over to the Elements panel and drag the smaller rectangle upon the other one. This way it will be placed within the larger element in regards to the DOM structure. Therefore they form a group and adopt animations for both elements at the same time.
