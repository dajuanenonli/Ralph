Recommendations

In this section we have included some links to helpful resources for surviving the modern web. It is far from being complete, but it should set you off in the right direction.

The more you know...

This publication is focused on Edge Animate. Yet it is important and very useful to understand the context in which Animate compositions reside. Furthermore, this introduction can only give you a glimpse on what there is to know. And as we have already learned, it is all a (fast) moving target. Therefore we have included a couple of links and recommended resources that should help you keep track on the development and guide you into this fascinating world called modern web.
