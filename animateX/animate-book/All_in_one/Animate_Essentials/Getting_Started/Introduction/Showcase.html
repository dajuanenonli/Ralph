Edge Animate is a very versatile tool and especially the context of the modern web, e-publications and applications grants a huge playground for interactive and engaging content. Here are some examples:

Shop Teaser

This demo illustrates how you can use Edge Animate to create interactive elements, for example as a banner for a website.

Interactive components

You can also create interactive elements and visual eye candies for websites, apps and e-publications.

Video player
