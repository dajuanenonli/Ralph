Edge Commons Spotlight Overlay

Lightbox effect with the Dirty little helpers

In this section you will learn how to use the Edge Commons Spotlight Feature to to dislplay images, videos and Animate composition in a lightbox like overlay. In all three cases, the basic principle is the same. You only have to modify the configuration object, which we will have a closer look at in this section further below. The spotlight overlay can be used to set media content into the right spot and is an easy to use improvement for all sorts of projects: websites, banners, interactive storytelling... you name it!

Spotlight Module

The Spotlight Overlay is available with the EC.Spotlight module. It is not part of the core features, therefore you have to either use the All-in-one package of the Edge Commons or additionally load this module together with the core.

Open a spotlight overlay

EC.Spotlight.open();
This function opens the spotlight overlay. It expects the configuration object, which defines the media type, source, size and other settings. Usually you would save this object right before calling the function and use the variable config to hand it in. The second, optional parameter allows you to set the context for the overlay, for example window.document or window.parent.document.

Example: Opening an image in a Spotlight Overlay

This example illustrates how you can use the Spotlight feature to open an image in an overlay. The config object contains the appropriate settings. Some parameters are mandatory, for example source and type, whereas borderWidth and borderColor are optional. 

          
// Define configuration for overlay in config object
var config = {
      width: 700,                                   // Width of the overlay	
      height: 400,                                  // Height of the overlay
      borderWidth: 5,                               // Border width (px)
      borderColor: "#FFF",                          // Border color
      type: "image",                                // Media type (image|animate|youtube)
      source: "images/MyImage.jpg",                 // Source to media
}

// Open the Spotlight Overlay with settings stored in the config object
EC.Spotlight.open( config );

Click on the button in the demo project below to see the image being opened in the Spotlight Overlay.

Example: Opening a Youtube video in a Spotlight Overlay

This example explains how you can use the Spotlight feature to open a Youtube Video in the Spotlight Overlay. The config object contains the appropriate settings. With the param object you can adjust additional parameters, for example autoplay. The source in this case is the Youtube ID for the video. 

          
// Define configuration for overlay in config object
var config = {
      width: 700,                                   // Width of the overlay	
      height: 400,                                  // Height of the overlay
      borderWidth: 5,                               // Border width (px)
      borderColor: "#FFF",                          // Border color
      type: "youtube",                              // Media type (image|animate|youtube)
      source: "1woru3cyFiw",                        // Source to media
      param: {                                      // Additional parameter
        autoPlay: true
      }
}

// Open the Spotlight Overlay with settings stored in the config object
EC.Spotlight.open( config );

Example: Opening an Animate composition in a Spotlight Overlay

This example demonstrates how you can use the Spotlight feature to open another Animate composition within the Spotlight Overlay. The config object contains the appropriate settings. Make sure to include this additional composition on file level into your project folder, in this case it is located in a folder called composition. The source in this case is the HTML file of that targeted additional composition. 

          
// Define configuration for overlay in config object
var config = {
      width: 700,                                   // Width of the overlay	
      height: 400,                                  // Height of the overlay
      borderWidth: 5,                               // Border width (px)
      borderColor: "#FFF",                          // Border color
      type: "animate",                              // Media type (image|animate|youtube)
      source: "composition/index.html",             // Source to media
}

// Open the Spotlight Overlay with settings stored in the config object
EC.Spotlight.open( config );

Close an existing spotlight overlay

EC.Spotlight.close();
This function usually gets called internally by the close button of the Spotlight Overlay automatically. You can however define a callback function within the config object to react to the closing of the Overlay. This allows you for example to pause and resume playback of the main composition when the Spotlight is opened.

Example: Resume playback when Spotlight is opened

In this example we want to pause our main composition when the Spotlight is being opened and resume playback when it is closed again. Therefore we add a callback to our config object. 

          
// Define configuration for overlay in config object
var config = {
      width: 700,                                   // Width of the overlay	
      height: 400,                                  // Height of the overlay
      borderWidth: 5,                               // Border width (px)
      borderColor: "#FFF",                          // Border color
      type: "image",                                // Media type (image|animate|youtube)
      source: "images/MyImage.jpg",                 // Source to media
      onClose: function(config) {                   // Callback for close button
        EC.unpause(sym);
      } 
}

// Pause Playback for the main composition
EC.pause(sym);
      
// Open the Spotlight Overlay with settings stored in the config object
EC.Spotlight.open( config );
