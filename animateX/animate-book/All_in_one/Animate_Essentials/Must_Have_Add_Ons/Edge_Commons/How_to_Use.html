How to use the Edge Commons

A quick guideline

Edge Commons is an extension library for Edge Animate. As such it needs to be integrated into the project in order to provide its functionality. This short section will give you an introduction into where you can find this library, how to add it and which versions are currently available. It will help you get set up and ready with the dirty little helpers, which will then be presented in a detailed way in the subsequent section. But first things first...

Where do I find the Edge Commons?

The Edge Commons project itself is hosted on GitHub. If you want to contribute and extend the library you should pay a visit there.
For working with the Edge Commons and to get a direct access to the files, you can either download the library – in order to use it as a local version – or use the online version via public server (CDN). You will find both possibilities and links on http://cdn.edgecommons.org or use the link here:

»Edge Commons 2.0«

Get the latest version of the Edge Commons (2.0.0) now:

Why are there different versions?

The Edge Commons are constantly growing and work in progress. In order to provide compatibility with different Animate releases and updates as well as projects, there are different versions of the Edge Commons available. Also, you won't necessarily need all the functionality in all of your projects. That's why the Edge Commons can also be divided into different modules.
Make sure to get the right version for your project.

jQuery dependency

Note that with the Edge Animate release of October 2014, jQuery is no longer a part of the Edge Animate core. Earlier versions of the Edge Commons still require jQuery, so you might have to load that library as well. With the release of Edge Commons 2.0.0, we've got rid of those dependencies so that you don't need jQuery anymore.

Modularity

The Edge Commons library consists of several packages. If you are new to Edge Commons we recommend the “All-in-one” package. This package includes all available features (except the experimental ones). If you deploy your final compositions you should consider loading only the modules that you actually use.

Distribution

You can either use a local version of the Edge Commons library, which means that you have to download the files and include them into your project folder. The other option is to use an online version and simply include a link to the CDN (content delivery network). Note that in this case the Edge Commons only work when you are online!

How do I include the Edge Commons into my project?

No matter which version of the Edge Commons you use, you have to include the library into your project in order to get access to the functionality. Since the update to Animte CC (3.0) in January 2014, you can use the library panel in Animate to add the Edge Commons library easily.

Step by step instruction:

  1. Go to http://cdn.edgecommons.org and choose your package.
    (If you are not sure which suits your needs, choose "All-in-One").




  2. In Edge Animate click on the "+" icon in the scripts panel of your library. Select "from URL" for the CDN version of Edge Commons. Otherwise choose "from Disk" if you have downloaded the library.




  3. Now Animate asks for the location/URL of the external library. Paste the URL from the CDN server or the path to the file in your file system.




  4. The Edge Commons library will now appear in the scripts panel. Now you can start using all the cool Edge Commons features in your composition.




Caution: Keep in mind that if you are in fact using an older version of Edge Commons and need to load jQuery as well, that you will have to load jQuery before the Edge Commons in order to make it work. Check the order in your scripts panel to be safe: jQuery should be listed above the Edge Commons.

Links

You can find the latest Edge Commons release (2.0.0) currently at Github right here:

  • https://github.com/simonwidjaja/EdgeCommons/tree/nojquery

If you want to use the older version and jQuery, you can use the links to the CDN versions of jQuery 2.1.1 and (the older) Edge Commons 1.4.0 with these URLs:

  • https://code.jquery.com/jquery-2.1.1.min.js
  • http://cdn.edgecommons.org/an/1.4.0/js/min/EdgeCommons.js
