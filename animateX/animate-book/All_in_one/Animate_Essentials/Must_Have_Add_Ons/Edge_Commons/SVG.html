Edge Commons Interactive SVG

A very promising little helper

SVG is a very promising image format for web. Not only can it be scaled without quality loss, it also allows you to place JavaScript inside of the image and thus making it interactive. This then allows you for instance to trigger events and send or retrieve information.

Currently Edge Animate only supports SVG files within <img> tags or as background images in <div> containers. So you can neither listen for such SVG events nor reach into the SVG structure to e.g. change the color of a shape. The "interactive SVG" feature provided by Edge Commons allows you to enable this in particular. The best thing about this is that you of course can still use the full potential of Edge Animate to stage your choreographies and create engaging interactive animations.

This feature requires jQuery to be loaded. Get the latest jQuery library here: http://code.jquery.com

Interactive SVG Module

The Interactive SVG feature is available with the EC.SVG module. It is not part of the core features, therefore you have to either use the All-in-one package of the Edge Commons or additionally load this module together with the core.

Convert SVG to be accessible for scripting

EC.SVG.accessSVG();
This function allows you to access the SVG document. The function expects a reference to the element that contains the SVG and then returns a corresponding object.

Example: Listen for custom event dispatched by the SVG

The following code snippet enables the interactive SVG feature on an SVG asset named "character" (which is placed on the stage). With the done function you are able to listen for the activation process to be completed and you get a reference to the actual SVG document passed in as a parameter (svgDocument). This reference can then be used to add event listener or change the SVG content (shapes, paths etc.) programmatically. 

          
// Enable SVG access
EC.SVG.accessSVG(sym.$("character")).done(
	function(svgDocument){
		
        // add event listener
		svgDocument.addEventListener("select", function(event) {
 
			// Remember selected part
			sym.setVariable("selectedPart", event.target);
 
			// show the id of the selected part in the textfield
			sym.$("selectedPartTxt").html( event.target.id );
		});
});

This SVG interactivity is very powerfull and can be used to create entire interfaces. This demo shows how you can use this feature to create a configurator. Clicking on the different parts of the image reveals the path that is being selected in the text field. You can then use the colors on the right side to change the color of that selected path in the SVG.
