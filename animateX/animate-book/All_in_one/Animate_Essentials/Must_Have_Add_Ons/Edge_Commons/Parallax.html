Edge Commons Parallax Scrolling

The most popular feature

In this section we will introduce you to the most popular feature of the Edge Commons library, our Parallax Scrolling. The basic idea here is to map the timeline onto the scroll bar of the browser, so that scrolling in the browser moves the playhead back and forth on the timeline - enabling you to create a parallax effect.

Usually creating such an effect takes a lot of work and effort to get the timing right. The aim here was to make use of Animate as a visual tool to help staging the choreography. At the same time we wanted to keep the required custom code to a bare minimum. Therefore we designed this Edge Commons feature to do all the heavy lifting in the background.
As a result you only need to include ONE single function into your project to make the parallax work.

So here's what you need to know...

Parallax Module

The Parallax Scrolling is available with the corresponding EC.Parallax module. It is not part of the core features, therefore you have to either use the All-in-one package of the Edge Commons or additionally load this module together with the core.

Setup Parallax Scrolling

EC.Parallax.setup();
This function allows you to enable the parallax scrolling feature for a specific composition. You have to hand in the reference to this composition (sym) in order to do so. Additionally, you can define the scroll type with a second argument - vertical or horizontal. The default is vertical.
Note that this simple Parallax Scrolling does not work on mobile devices. Check out the UBR.Parallax for a more advanced Parallax Scrolling that even performs great on mobile devices.

Example: Setting up a simple vertical Parallax Scrolling

This example shows you how to set up the Parallax feature for a vertical scrolling effect. Note that you have to use this script in compositionReady and NOT in creationComplete. 

          
// Setup Parallax Scrolling
EC.Parallax.setup(sym,"vertical");

You can scroll up and down in this demo composition to see the Parallax at work.
