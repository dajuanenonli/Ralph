Dirty little helpers for Edge Animate

Edge Animate is a very versatile tool. The openness to custom code and external libraries make it even more interesting. Yet there are some occasions where customizing the project requires quite a huge load of code and JavaScript skills. Especially for designers and beginners, this can soon become quite the obstacle.

This is where the Edge Commons come into play: They provide easy-to-use functions and simple solutions for complex problems in the Edge Animate universe. They include a whole range of different features to enhance the potential of Edge Animate â€“ such as parallax scrolling or the spotlight overlay. They also support you with more basic enhancements in form of helper functions, for example to keep elements centered in flexible layouts.
No matter if you are using a complex feature or smaller helper functions - these dirty little helpers will boost your performance and bring working with Animate to a whole new level, while keeping the amount of code required to a complete minimum. Awesome, isn't it?!?

The general concept

Edge Commons, also known as "Dirty little helpers for Edge Animate", is an open source project that has been initiated in order to enhance the functionality and provide easy solutions for complex problems in Animate projects. There are a couple of useful JavaScript and jQuery libraries out there, but the Edge Commons are dedicated to Edge Animate and the first, largest and most frequently used library in this regard.

The content of the Edge Commons has derived from actual problems we have encountered throughout our countless experiments and projects as well as requests from the community. The initial idea was to analyze and collect the most frequent issues as well as the most interesting concepts to create a library with useful solutions, that requires as little effort and knowledge as possible.

The mission is simple: Making Edge Animate better, more accessible for especially for designers and a lot more fun to work with. Since it is an open source project, anybody can join in and contribute. This does not mean that you have to write own features or any code at all (though this would be very much appreciated) - we are also happy about people willing to test new features, create examples or simple hand in more feature requests.
